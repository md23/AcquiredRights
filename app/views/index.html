Prototype v0.1

This prototype simulate 3 journeys to be taken by Acquired Rights internal portal users

  • Case worker main journey
    User: Case worker
    On this journey the case worker assigns themselves to an application, take notes and submit decision outcome.

  • Appeal journey
    User: Authorised Appeal user
    On this journey an authorised Appeal user can assign themselves to a declined application that has been appeal and submit a decision outcome.

  • Reassigning application journey
    User: Case worker
    On this journey a case worker can reassign an application owned by a colleague to themselves.
